The MIUI ROM

Jul 26th, 2011

Just installed the MIUI ROM on my Android phone. I’d rooted my phone about 3 days after purchasing it and had been running CyanogenMod 7. CyanogenMod is a nice, stable ROM for people who want a no frills Android version which is closest to the stock Android distribution with a few improvements. On the other hand, the MIUI ROM contains lots of improvements over the stock distribution. The stock Android interface has been given a complete overhaul with custom theme elements that can be applied to almost every aspect of the UI (notification area, lockscreen image etc). My biggest gripe with CyanogenMod was the ADW launcher which would randomly disappear at times after I’d unlocked the phone and swiped somewhere on the homescreen. I believe this was a feature in ADW launcher but it annoyed me to great lengths as I’d end up losing the dialler application from the homescreen. The MIUI ROM has an app launcher of its own that is probably a ripoff inspired by the iPhone with apps spread across multiple home screens. This is quite convenient and I find it more intuitive compared to the app drawer interface (ADW launcher/stock android launcher). A nice feature is a backup application built right out of the box that can do local and remote backups (I’ve not tested this entirely). The ROM also detects an Apps2SD partition and moves applications to the SD card if required. This is a huge plus for people who have phones with limited internal memory and small /data partitions (e.g. the HTC Desire). I was surprised to find that even with all UI effects, the phone is still very responsive and seems to feel faster. The only minor issues I’ve faced so far are strange translation bugs (the ROM is originally Chinese with an added English language pack) and a missing weather application. The ROM is updated every Friday so the translation bugs should be fixed very soon. This is a really promising ROM that I hope to use for a long time.

Getting SmartVOIP to Work on CM7

Jul 24th, 2011

Android 2.3 (Gingerbread) claims to support SIP natively. For some weird reason it does not play well with SmartVOIP, probably due to missing STUN server support. SmartVOIP provide a native android app called MobileVOIP but it doesn't integrate with the Android UI. I found this nifty little app called CSipSimple(http://code.google.com/p/csipsimple/) which has loads of configuration options (e.g. connect when on WiFi and or 3G etc). It integrates really well with the native CyanogenMod UI and seems to be working really well in my limited tests. It's available on the android market by the same name. Installing Rubygems in Your Home Folder

Jul 16th, 2011

No root? No problem! Obtain a copy of rubygems from ruby forge.

$ tar xzvf rubygems-1.4.2 
$ cd rubygems-1.4.2 
$ ruby setup.rb --prefix=/your/home/folder/local 
$ echo "RUBYLIB=$HOME:/same/path/as/above/lib" >> ~/.bash_profile 
$ echo "export RUBYLIB" >> ~/.bash_profile 
$ source ~/.bash_profile

This should give you a working install of rubygems. Check out the environment

$ gem env

Install something

$ gem install rake

Enjoy!

UltraVPN on Lucid Lynx

Aug 31st, 2010

I was trying to get UltraVPN to work on linux using the instructions found at here but I had no luck at all since the configuration files mentioned in the post are inaccessible. To get UltraVPN to work, the following steps worked for me. # sudo apt-get install network-manager-openvpn Next, grab a copy of the windows installer from the website named “ultravpn-install.exe”. To get the configuration files, open the file “ultravpn-install.exe” with archive manager. Extract the config folder somewhere (the location is not important). Click on the network icon on the systray and go to VPN connections -> Configure VPN. Select “Import” and provide the path of the client.ovpn file inside the config folder. Set the username and password accordingly (as registered on the UltraVPN website). You might want to restart network manager for it to work. #sudo service network-manager restart After this, you should have an option under VPN connections to connect to UltraVPN (or whatever you named it during the import). Click on it to connect.
